Homework 8: Geoprocessing Analysis

My map answers the following question: "How many bus stops exist within a one mile radius of a hosptial in Wake County?" To create this map, I first created a buffer with a one mile radius around all the points representing hosptials in Wake County, then created an intersection between the buffer and all the points representing bus stops so as to only show the bus stops within the one mile radius.
